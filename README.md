# Math Quiz Console Game (C++)

## Description

This is a **console-based Math Quiz Game** developed using **C++**. The game generates random math questions based on the selected difficulty level and operation type. The player answers a series of questions and receives a final result (PASS or FAIL) based on their performance.

This project was built to practice:

* Core C++ concepts
* Enums and structs
* Functions and modular design
* Random number generation
* Console input/output

---

## Features

* Choose number of questions
* Multiple difficulty levels:

  * Easy
  * Medium
  * Hard
  * Mixed
* Multiple operation types:

  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Mixed operations
* Instant feedback for each answer
* Final quiz summary (score, level, operation type)
* PASS / FAIL result
* Play again option
